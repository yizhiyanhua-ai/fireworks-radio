# Legal and Content Boundary

## Read This First

`fireworks-radio` is a workflow tool. It is not a media rights grant.

This project can automate playlist building, caching, and playback. It does not
grant any license to music, podcasts, videos, streams, or any third-party media
that users choose to access.

## What the Project Permits

- Use of the software under the repository license
- Personal experimentation
- Noncommercial workflow automation, subject to the repository license

## What the Project Does Not Permit

- Commercial use under the default repository license
- Any implied right to reproduce, distribute, cache, or rebroadcast third-party media
- Any guarantee that a media source is legally reusable in your jurisdiction

## User Responsibility

Users are responsible for:

- verifying platform terms
- verifying copyright permissions
- deciding whether streaming is permitted
- deciding whether local caching or downloading is permitted
- deciding whether redistribution is permitted

## Practical Position

If a user uses this project to operate on copyrighted material without the
necessary rights, that risk belongs to the user and their workflow choices, not
to an implied media license from this repository.
