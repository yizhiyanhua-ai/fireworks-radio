# Image Prompts

## Why the Visuals Matter

This project is not a generic music player. Its visual system should communicate:

- fast fallback
- coding-session focus
- terminal-native pragmatism
- music energy without looking like a consumer streaming clone

## Open Source Project Icon

Square app icon. Bold terminal-wave emblem. A clean monoline waveform fused with a command prompt caret, centered inside a rounded square. Dark graphite background, electric orange and cyan accents, subtle glow, minimal but memorable, open-source devtool vibe, flat vector look, crisp edges, no text, macOS developer aesthetic.

## Landing Page Hero

Wide landing page hero for an open-source coding music fallback tool. A dark developer desk scene rendered as editorial product art: terminal panels, minimal waveform rails, playlist cards, muted code windows, orange-cyan focus lights, cinematic but clean, premium open-source startup aesthetic, strong negative space for headline, 16:9 composition, no watermarks, no text baked in.
